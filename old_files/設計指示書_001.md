# 🕹️ 探偵風ターミナルゲーム 設計指示書

## 📘 プロジェクト概要
このプロジェクトは **Linux コマンド学習用探偵風 CUI ゲーム** です。  
プレイヤーはターミナル（xterm.js）や GUI を操作し、ミッションをクリアしながら学習を進めます。

---

## 🧩 使用技術

### フロントエンド
- Vue.js / Nuxt4
- Tailwind CSS
- xterm.js（ターミナル UI）

### バックエンド
- Django + Django REST Framework（API）
- Django Channels（WebSocket）
- Python（正規表現・コマンド評価ロジック）

### 認証
- PyJWT（JSON Web Token 認証）

### コマンド評価
- シェル風評価器（擬似 shell evaluator）

---

## 🖼️ UI / UX
```

┌──────────────────────────────────────────────┐

│ 左上：日本語カレント表示                    │

│ 例：現在地：事務所                          │

├──────────────────────────────────────────────┤

│ メイン画面（背景画像 + 情報）                │

│ 右側：コマンド一覧                           │

│       → コマンド名クリックで詳細説明を表示   │

├──────────────────────────────────────────────┤

│ 下部：CUI ターミナル（xterm.js 表示）         │

└──────────────────────────────────────────────┘

```
- コマンド一覧はレスポンシブに 2〜3 列表示  
- コマンドクリックで詳細説明が **フェードイン** 表示

---

## 🧠 コマンド仕様

### 初級
- `cd`, `ls`, `pwd`, `touch`, `cat`, `less`, `echo`

### 中級
- `grep`, `find`, `sort`, `uniq`, `awk`

### 補助
- `history`, `clear`

---

## 🌐 正規表現仕様

- `grep` や `awk` などで正規表現のパターン検索を可能にする  
- 拡張正規表現 (`-E`) に対応  
- バックエンドで正規表現を評価し結果を返す

---

## 🔄 local と remote（ssh）挙動

### local_dir 構造
```

root/              ← 事務所

├ desk/

├ door/

└ park/

├ swing/

├ slide/

├ chan-up_bar/

├ seesaw/

└ jungle_gym/

```
※ 全て英語ディレクトリ名  
※ 日本語は GUI 表示補助用のみ

### remote_env 構造
- 遊園地、学校、博物館などは **remote（ssh 接続必須）**

---

## 🧠 ssh / exit 挙動

### 擬似 SSH
```

$ ssh amusement_park

```
- `remote_mode = True`  
- `current_root = "amusement_park_env"`  
- GUI は **背景フェード** で local → remote

### exit
```

$ exit

```
- `remote_mode = False`  
- `current_root = "root"`  
- GUI は **フェードで remote → local**

### cd は戻らない
- remote から local へ戻るには必ず `exit`

## ✨ WebSocket（xterm.js）

### フロー
1. xterm.js から入力を受け取る
2. Django Channels がテキストを受信
3. 受信したテキストを evaluator で判定
4. 結果を WebSocket で返す
