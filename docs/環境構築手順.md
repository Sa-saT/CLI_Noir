# CLI_Noir 環境構築手順（インストール済み前提）

このドキュメントは、`volta` / `pnpm` / `brew` / `pyenv` が既にインストール済みの状態から、  
CLI_Noir の開発環境を立ち上げるための実行手順をまとめたものです。

---

## 1. 前提

- OS: macOS
- シェル: zsh
- `volta`, `pnpm`, `brew`, `pyenv` はインストール済み

---

## 2. フロントエンド構築（Vue3 / Nuxt）

### 2-1. Node / pnpm バージョン確認
```bash
node -v
pnpm -v
volta list
```

### 2-2. Nuxt プロジェクト作成
```bash
pnpm create nuxt@latest frontend
cd frontend
pnpm install
```

### 2-3. 追加パッケージ導入（2026-07-06 技術選定）
```bash
pnpm add pinia @pinia/nuxt
pnpm add -D vitest @vue/test-utils @playwright/test eslint prettier
```

- TypeScript は Nuxt 標準のまま有効で使う（WS フレーム型を `types/ws.ts` に定義し設計指示書 § 7 と一致させる）
- `pinia`: state の単一ソース（`docs/DESIGN.md` § 10-1）
- `vitest` + `@playwright/test`: `docs/DESIGN.md` § 10-8 のテスト基準
- `nuxt.config.ts` に `ssr: false` を設定（SPA モード。設計指示書 § 2）

### 2-4. 開発サーバー起動
```bash
pnpm dev
```

---

## 3. バックエンド構築（Python / FastAPI）※2026-07-06 改訂（Django → FastAPI）

### 3-1. Python バージョン固定
```bash
pyenv local 3.12.8
python -V
```

### 3-2. 仮想環境作成と有効化
```bash
python -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
```

### 3-3. 依存パッケージ導入
```bash
pip install "fastapi[standard]" sqlmodel alembic pyjwt "passlib[bcrypt]" pydantic-settings
pip install pytest httpx ruff   # 開発用
```

- `fastapi[standard]`: uvicorn（ASGI サーバー）同梱。WebSocket ネイティブ対応
- `sqlmodel` + `alembic`: ORM（SQLAlchemy ベース）とマイグレーション
- `pyjwt` + `passlib[bcrypt]`: JWT の発行/検証とパスワードハッシュ
- `httpx`: API テストクライアント / `ruff`: リント + フォーマット

### 3-4. プロジェクト作成（FastAPI は生成コマンドがないため手で掘る）
```bash
mkdir -p backend/app/{api,ws,evaluator,models} backend/tests
touch backend/app/main.py backend/app/settings.py
```

推奨スケルトン:
```text
backend/
├ app/
│  ├ main.py       # FastAPI アプリ生成・ルータ登録
│  ├ settings.py   # pydantic-settings（.env 読み込み）
│  ├ api/          # HTTP: auth / missions / state（読み取りのみ）
│  ├ ws/           # /ws/terminal エンドポイント + フレームの Pydantic モデル
│  ├ evaluator/    # 擬似シェル（純粋 Python。FastAPI に依存させないこと）
│  └ models/       # SQLModel テーブル定義（User / MissionState）
├ alembic/         # マイグレーション
└ tests/           # pytest（ゴールデントランスクリプトテスト含む）
```

### 3-5. 起動確認
```bash
cd backend
alembic upgrade head
uvicorn app.main:app --reload --port 8000
```

- `http://localhost:8000/docs` で OpenAPI ドキュメントが表示されれば OK（FastAPI が自動生成）

## 5. 動作確認チェック

### フロント
- `pnpm dev` で起動できる
- `/missions` ルートへアクセスできる

### バック
- `uvicorn app.main:app --reload` で起動できる
- `/docs`（OpenAPI）に `/api/auth/login/` と `/api/missions/` が表示される
- `/ws/terminal` に WebSocket 接続でき、`auth` → `hello` が返る

---

## 6. トラブルシュート

- `ModuleNotFoundError`
  - 仮想環境有効化（`source .venv/bin/activate`）後に `pip install` を再実行する

- Python バージョンが想定と異なる
  - `pyenv local 3.12.8` を再実行し、`python -V` を確認する
